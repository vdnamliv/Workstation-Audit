🔧 Component Architecture

Frontend Layer

Dashboard (Alpine.js SPA)

  • Real-time policy editing với syntax highlighting
  • Live agent status monitoring
  • Interactive compliance dashboards
  • Multi-format report generation

API Gateway Layer

Nginx Reverse Proxy

  • mTLS certificate validation
  • SSL termination
  • Load balancing và routing
  • Rate limiting và security headers

Backend Services

VT-Server Backend (Go)

  • Dashboard Handler (:8080)
  • Agent Handler (:8081)
  • Enrollment Gateway (:8082)
  • PostgreSQL integration

Policy & Audit Execution

  1. Agent requests policies via mTLS authentication
  2. Server returns latest policy YAML
  3. Agent executes audit locally (registry, files, services)
  4. Agent submits results to server
  5. Server processes và stores trong PostgreSQL

🔐 Security Architecture

🛡️ Authentication

  • mTLS với Step-CA certificates
  • OIDC với Keycloak
  • X-Test-Mode fallback
  • JWT token validation

🔒 Certificate Management

  • Automatic enrollment
  • 24-hour certificate lifetime
  • Auto-renewal trước expiry
  • Revocation support
